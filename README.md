# 🚀 Serverless Task Manager

A full-stack **serverless task management application** built with **React, AWS Lambda, API Gateway, MongoDB Atlas, and Amazon S3**.

---

## 🌐 Live Application

Frontend (Hosted on Amazon S3)

http://taskmgr-frontend.s3-website-us-east-1.amazonaws.com/login

---

## 🔗 Backend API (AWS API Gateway)

Example endpoints:

POST /api/auth/login
POST /api/auth/register
GET /api/tasks
POST /api/tasks/create

Base API URL

https://3qk08knqsb.execute-api.us-east-1.amazonaws.com
